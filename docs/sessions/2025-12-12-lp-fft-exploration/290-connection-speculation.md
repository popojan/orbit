## Observation

The γ-framework uses two dual frameworks with different multiplication offsets:

| Framework | Offset | Multiplication |
|-----------|--------|----------------|
| A | 5/4 | t₁ ⊗ t₂ = t₁ + t₂ + 5/4 |
| B | 7/4 | t₁ ⊗ t₂ = t₁ + t₂ + 7/4 |

For n-th roots of unity, the first root (k=1) is:
```
ρ[n, 1] = 2/n - 5/4 = (8 - 5n) / (4n)
```

The **numerator** `5n - 8` generates an arithmetic progression.

## Key Observations

### 1. Disjoint Coverage

The sets are **completely disjoint** - no critical integer appears in both sequences:
- 5n-8 hits: {2, 7, 17, 22, 37, 42}
- 7n-8 hits: {6, 13, 34}

### 2. Small n Values

All hits occur for small n values (n ≤ 10), suggesting the connection is with "small" roots of unity.

### 3. Framework Duality

Frameworks A and B are related by `CircDual[t] = 1 - t`, which swaps the offset:
```
1 - 5/4 = -1/4 ≡ 7/4 (mod 2)
```

The disjoint nature of the hits might reflect this duality.

### 4. Missing Critical Integers

The 20 critical integers NOT covered by either framework:
```
{1, 3, 5, 10, 14, 15, 19, 21, 23, 26, 29, 30, 31, 35, 58, 93, 110, 145, 203, 290}
```

Note: 290 itself is NOT expressible as 5n-8 or 7n-8 for integer n:
- (290 + 8) / 5 = 298/5 (not integer)
- (290 + 8) / 7 = 298/7 (not integer)

## Speculative Hypotheses

### Hypothesis 1: Extended Framework Family

Perhaps there's a family of frameworks with offsets (2k+1)/4 for k = 2, 3, 4, ...:
- k=2: 5/4 (Framework A)
- k=3: 7/4 (Framework B)
- k=4: 9/4
- etc.

**Question:** Does the union of all (2k+1)n - 8 sequences cover more critical integers?

### Hypothesis 2: Quadratic Form Connection

The 290-theorem concerns **quadratic forms**. The γ-framework parametrizes the **unit circle** (a quadric).

**Question:** Is there a deeper connection between:
- L^p geometries (different "unit circles")
- Quadratic form representability
- The critical set {1, 2, ..., 290}?

### Hypothesis 3: Escalator Lattice Dimensions

In the 290-theorem proof, escalator lattices have dimensions 1-7:
- Dimension 1-2: few truants
- Dimension 3-4: most truants appear
- Dimension 5+: only 203, 290

The n values we found (2, 3, 5, 6, 9, 10 for Framework A) might relate to these dimensions.

### Open Question: Transitive Closure vs Multiple Pairs

Two approaches to cover all 29 critical integers:

**A) 3 disjoint pairs:**
Each pair `(2k+1)/4 ↔ (2k+3)/4` covers ~31%, need 3 pairs for ~93%

**B) Transitive closure over 2 groups + transformation T:**
```
G₁ = {2,7,17,22,37,42}  (from 5n-8)
G₂ = {6,13,34}          (from 7n-8)
T(G₁ ∪ G₂) → more critical integers
```

Preliminary finding: `T = n+8` hits 4 missing integers {14,15,21,30}.
This might not be coincidence since our formulas are `an-8`.

**Constraint:** Max 3-4 groups total (Occam's razor).
